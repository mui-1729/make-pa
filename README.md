# Make PA

PA表PDFをアップロードし、PDF内のテキストをブラウザ上で編集して、シンプルなPA表PDFとして再出力するMVPです。

## プロジェクト構成

```text
.
├── app/
│   ├── globals.css
│   ├── layout.tsx
│   └── page.tsx
├── package.json
├── tsconfig.json
└── README.md
```

最初のプロトタイプなので、PDF読み取り、元PDF上への手動記入欄配置、ステージ図、PDF出力の主要ロジックは `app/page.tsx` に集約しています。

## 使用ライブラリ

- Next.js / React / TypeScript: ローカルWebアプリの基盤
- pdfjs-dist: PDF内テキストの抽出
- html2canvas: 画面上のPA表プレビューの画像化
- jsPDF: 画像化したPA表のPDF保存

日本語PDF出力のフォント埋め込みを避けるため、DOMを画像化してPDFに貼り付ける方式にしています。MVPとして見た目を保ちやすい実装です。

## 起動方法

```bash
npm install
npm run dev
```

起動後、ブラウザで次を開きます。

```text
http://localhost:3000
```

## 使い方

1. トップ画面のPDFアップロード欄からPA表PDFを選択します。
2. PDF内のテキストが「読み取ったPDF内容」に表示されます。
3. 「元PDF上の記入欄」で「記入欄を追加」を押します。
4. 追加した記入欄のラベル、内容、ページ、横位置、縦位置、幅、高さを調整します。
5. ステージ図で、マイク、アンプ、人、DI、モニター、ドラムを追加します。
6. 追加したパーツをステージ上でドラッグして配置します。
7. 出力プレビューを確認し、「PDFとしてダウンロード」を押します。

アップロードしたPDFがある場合は、「元PDFテンプレを保持」が自動でオンになります。この状態では、元PDFページを背景として表示し、ユーザーが追加した記入欄を同じページ上に重ねます。PDF出力時も、元PDFの枠やレイアウトを背景として残したまま、入力した内容を重ねます。

## 元PDF上の記入欄

自動推定は使わず、元PDFの枠を見ながら必要な記入欄を手動で配置します。PA表ごとに枠位置や項目名が違う前提で、テンプレートの見た目を崩さないことを優先しています。

- 項目名は編集できます。
- 記入内容は複数行で入力できます。
- ページ、横位置、縦位置、幅、高さを数値で調整できます。
- 入力欄は必要な分だけ追加できます。
- PDFプレビュー上にも同じ入力欄が表示されます。

## ステージ図機能

編集画面の「ステージ図」から、以下のパーツをボタンで追加できます。

- マイク
- アンプ
- 人
- DI
- モニター
- ドラム

追加したパーツはステージ図上でドラッグ移動できます。選択中のパーツは「選択を削除」で削除でき、「全消去」でステージ図を空にできます。作成したステージ図はPDF出力プレビューにも反映されます。

## 試作用テンプレート

動作確認用に、テキスト抽出できるPDFを `samples` に入れています。

- `samples/pa-template.pdf`: 空欄のPA表テンプレート
- `samples/pa-sample-filled.pdf`: 入力済みのPA表サンプル

まずは `samples/pa-sample-filled.pdf` をアップロードすると、抽出テキストの表示を確認しやすいです。

## 現時点の制限

- 画像PDFのOCRには対応していません。
- 入力欄のドラッグ移動・マウスリサイズは未実装です。現時点では数値入力で位置とサイズを調整します。
- ステージ図は簡易配置のみで、回転、サイズ変更、ラベル編集には未対応です。
- PDF本文の抽出テキストは確認用として残しています。
